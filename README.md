# Billiard Management System

Sistem manajemen untuk aplikasi penyewaan meja biliar dengan fitur booking, payment, dan management user.

## Fitur Utama

- Manajemen meja biliar
- Booking meja biliar
- Payment processing
- User management
- Laporan dan statistik

## Tech Stack

- Laravel (Backend)
- Vue.js/React (Frontend)
- MySQL (Database)
- Vite (Build tool)
